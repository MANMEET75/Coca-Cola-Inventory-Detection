# Coca Cola Inventory Detection

## Steps to Run it
```bash
git clone https://github.com/MANMEET75/CokeDetectionAI.git
```

```bash
conda create -n venv python==3.11 -y
```

```bash
conda activate venv
```


```bash
pip install -r requirements.txt
```

```bash
streamlit run app.py
```

Enjoy Coding!!
